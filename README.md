# 포트폴리오 AI 분석 봇

스크린샷 한 장으로 투자 포트폴리오를 자동 입력하고, AI가 리밸런싱 조언을 해주는 도구입니다.

---

## 파일 구조

```
portfolio/
├── .env                  # API 키 (외부 공개 금지)
├── portfolio.xlsx        # 포트폴리오 데이터 (메인 파일)
├── create_template.py    # 최초 1회: 엑셀 템플릿 생성
├── image_to_excel.py     # 스크린샷 → 엑셀 자동 입력
└── portfolio_reader.py   # 엑셀 읽기 + 분석 리포트 + AI 프롬프트 생성
```

---

## 처음 시작할 때 (최초 1회)

### 1. 패키지 설치
```bash
pip install google-genai python-dotenv openpyxl pandas Pillow
```

### 2. API 키 설정
`.env` 파일에 Gemini API 키 입력
```
GEMINI_API_KEY=발급받은_키_입력
```
> Gemini API 키 무료 발급: https://aistudio.google.com

### 3. 엑셀 템플릿 생성
```bash
python3 create_template.py
```
`portfolio.xlsx` 파일이 생성됩니다. 예시 데이터가 들어있으니 실제 본인 데이터로 교체하세요.

---

## 매번 사용할 때

### 방법 A — 스크린샷으로 자동 입력 (추천)

**1단계:** 뱅크샐러드 또는 토스에서 자산/투자 화면 스크린샷 찍기

**2단계:** 스크린샷을 `portfolio` 폴더에 저장

**3단계:** 자동 입력 실행
```bash
python3 image_to_excel.py 스크린샷파일명.png
```
예시:
```bash
python3 image_to_excel.py toss_20260302.png
```
- 기존 종목이면 현재가/수량 자동 업데이트
- 새 종목이면 자동 추가

**4단계:** 분석 리포트 확인
```bash
python3 portfolio_reader.py
```

---

### 방법 B — 엑셀 직접 입력

`portfolio.xlsx` → `포트폴리오` 시트에 직접 수정 후 아래 실행:
```bash
python3 portfolio_reader.py
```

## 엑셀 입력 규칙

| 컬럼 | 설명 | 예시 |
|------|------|------|
| 종목명 | 종목 또는 자산명 | 삼성전자, 비트코인, VOO |
| 자산유형 | 아래 6가지 중 하나 | 국내주식 |
| 플랫폼 | 보유 증권사/거래소 | 키움증권, 업비트 |
| 통화 | KRW 또는 USD | USD |
| 수량 | 보유 수량 (소수점 가능) | 0.05 |
| 매입단가 | 평균 매입단가 (해당 통화 기준) | 65000 |
| 현재가 | 현재 가격 (해당 통화 기준) | 72000 |
| 매입금액(원) | 비워두면 자동계산 / USD 자산은 원화 환산값 직접 입력 권장 | |
| 메모 | 자유 메모 | 헤지용 |

**자산유형 종류:** `국내주식` / `해외주식` / `국내ETF` / `해외ETF` / `가상자산` / `현금성`

## 환율 변경

`portfolio_reader.py` 상단의 값을 수정:
```python
USD_TO_KRW = 1350.0  # 현재 환율로 변경
```

## 주의사항

- `.env` 파일은 절대 외부에 공유하지 마세요 (API 키 포함)
- USD 자산의 평가금액은 환율 적용 기준이므로 환율 업데이트 필요
- 스크린샷 여러 장을 순서대로 실행하면 누적 업데이트됩니다
